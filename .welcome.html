HLD Filter

Description:
A unix based mail filter written in perl to replace procmail and many other mail utilities. Source code is free and released under GNU GPL.
Version 2.4 brings HLDFilter into a swiss army knife type of utility. Because it is written entirely in perl, and has many features, hldfilter is not the fastest mailfilter, but for 95% of users, this is not an issue anyways. Please read the Features section for more info.

Features:

  • Stats - Track who emails you most, or spams you most. Displayed in HTML with graphs.
  • Mailback's - Setup auto replies to keywords to respond with things like your uptime, or your pgpkey. (both commands and files are supported)
  • Spam rejecting or ignoring
  • Reporting spammers to their ISP's automatically
  • Mail lists - Keep emails in different folders automatically
  • Real Time Blackhole checks
  • Verifies the validity of a senders email address
  • Forwarding of emails to other email addresses
  • Advanced logging
  • Command-line configuration
  • Command-line processing of messages
  • Easy to use and install

Future plans:
  • Add the ability to process procmail log files for webstats
  • Fix "Date" header in emails to not be any time later than the current time.
  • Add ability to capture all attachments into a specified directory
